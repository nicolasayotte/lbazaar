**Finding:** `Discord::buildExchangePointsMessage()` sent Discord webhook notifications for legacy points-to-NFT exchange requests. The method and its `'exchange'` entry in `$buildMethods` were removed. The calling code in `ProfileController::exchangeToNFTRequest()` and `ProfileController::feedPointsToWallet()` were also dead code (no routes pointed to them) and were removed along with their unused imports and properties.

**Files changed:**
- `app/Classes/Discord.php` — removed `buildExchangePointsMessage()` method and `'exchange'` map entry
- `app/Http/Controllers/Portal/ProfileController.php` — removed `feedPointsToWallet()`, `exchangeToNFTRequest()`, unused imports (`Discord`, `Validator`, `WalletService`, `EmailService`, `Request`, `Http`, `Log`), and unused `$walletService`/`$emailService` properties

## Acceptance criteria	

We will update to:
* Modify the prices of classes to be in ADA
* Remove UI related to points
* Implement the designed UI for the NFT minting for class completion
* Implement the designed UI for token rewards
* Verify / fix any wallet flow
* Implement Stripe for credit card payments for class purchases (Japan-compliant implementation), including integration + testing + documentation.

Product demo video includes the user experience like below:

* Students can buy classes with ADA
* Students can buy classes using credit card via Stripe
* Students can receive NFTs as completion of the class (if teachers set it)
* Students can receive tokens as a reward of completion of the class (if teachers set it)
* Teachers can set the NFT as completion rewards
* Teachers can set token rewards
